# Support Ops Environment

A reinforcement-learning environment simulating SaaS support operations.

## Features
- Stateful queue system
- Tool budget constraints
- Incident propagation
- Policy conflict engine

## Tasks
1. Triage Sprint (easy)
2. Queue Under Pressure (medium)
3. Incident Cascade (hard)

## Setup
```bash
uv sync
uv run server
```

## Inference
```bash
python inference.py
```
